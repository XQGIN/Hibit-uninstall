# HiBit Uninstaller 使用说明与卸载指南


---

## 软件简介

HiBit Uninstaller 是一款轻量、免费、无广告的 Windows 平台第三方卸载工具。相比系统自带的“应用和功能”，它支持**强制卸载、深度残留扫描、浏览器扩展管理、上下文菜单清理、开机启动项管理**等功能，适用于彻底移除顽固软件、流氓程序或已损坏的应用。

## 核心操作指南

### 1. 标准卸载（推荐）

适用于正常安装且卸载程序完整的软件。

1. 启动 HiBit Uninstaller，主界面自动扫描已安装程序。
2. 在列表中找到目标软件，勾选复选框。
3. 点击顶部工具栏 `卸载` 按钮。
4. 按原软件卸载向导完成操作。
5. 原卸载结束后，HiBit 会**自动弹出残留扫描窗口**，勾选所有扫描到的注册表项、临时文件、空文件夹，点击 `删除选中项` 即可。

### 2. 强制卸载（Force Uninstall）

适用于：

- 软件卸载程序已丢失或损坏
- 常规卸载卡死/报错
- 怀疑为捆绑/残留组件

**操作步骤：**

1. 点击左侧导航栏 `强制卸载`。
2. 输入软件名称或粘贴安装路径（如 `C:\Program Files\Example`）。
3. 点击 `扫描`，工具将分析相关文件、注册表及服务项。
4. 核对扫描结果（⚠️ 勿勾选系统关键路径如 `Windows\System32`）。
5. 点击 `强制删除` 完成清理。
6. 建议重启计算机使更改生效。

### 3. 残留深度清理

- 左侧菜单 `磁盘清理`：扫描临时文件、更新缓存、回收站等。
- 左侧菜单 `空文件夹清理`：移除卸载后遗留的空目录。
- 左侧菜单 `浏览器扩展`：管理 Chrome/Edge/Firefox 插件，可一键禁用或移除。

## 如何卸载 HiBit Uninstaller 自身

### 方法 A：常规卸载

1. 打开 Windows `设置 > 应用 > 安装的应用`（或控制面板 `程序和功能`）。
2. 搜索 `HiBit Uninstaller`，点击 `卸载`。
3. 按提示完成即可。工具会在卸载前询问是否保留配置，按需选择。

### 方法 B：便携版（Portable）

若下载的是免安装便携版，直接**删除所在文件夹**即可完全移除，无注册表残留。

> 💡 提示：卸载后可手动检查 `C:\Users\你的用户名\AppData\Local\HiBit` 是否存在残留配置文件夹，按需删除。

## 注意事项与常见问题

### ⚠️ 安全警告

- 强制卸载可能误删共享组件，请仔细核对扫描结果。
- 不要清理名称含 `Microsoft`、`Windows`、`Driver` 的系统项。
- 涉及驱动、安全软件、系统优化工具时，建议优先使用其官方卸载程序。

### ❓ 常见问题

| 问题                       | 解决建议                                                     |
| -------------------------- | ------------------------------------------------------------ |
| 卸载后桌面图标未消失       | 手动删除快捷方式，或运行 `磁盘清理` 刷新缓存                 |
| 强制卸载后软件仍出现在列表 | 重启电脑后刷新列表；若仍存在，使用 `注册表清理` 搜索相关键值 |
| 扫描残留提示“访问被拒绝”   | 以管理员身份运行；关闭杀毒软件实时防护后重试                 |
| 误删了重要文件             | 立即停止写入操作，使用数据恢复工具（如 Recuva）尝试找回      |
